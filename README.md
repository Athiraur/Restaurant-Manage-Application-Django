# Title
Restaurant Manage Booking Application 

# Overview
A web-based system to manage hotel and restaurant bookings efficiently. Users can check availability and book rooms, while admins manage hotels, rooms, and bookings.

# Features
Admin & User Login
Hotel and Room Management
Booking & Availability Tracking
Payment Module

# Technologies
HTML, CSS, Bootstrap, JavaScript
Python, Django
SQLite3

# Run Project
git clone <repo-link>
cd project-folder
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
Open: http://127.0.0.1:8000/

# Future Scope
Payment gateway, notifications, mobile app
